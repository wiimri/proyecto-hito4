# Imagenes del prototipo

Coloca aqui tus imagenes para usarlas en publicaciones.

Ejemplo:

```text
client/assets/notebook.jpg
client/assets/bicicleta.png
client/assets/silla.webp
```

Luego, en el formulario de crear publicacion, escribe la ruta:

```text
assets/notebook.jpg
```

Tambien puedes usar el campo **Subir imagen** para seleccionar una imagen desde tu computador. Esa opcion guarda la imagen dentro del navegador usando `localStorage`.
